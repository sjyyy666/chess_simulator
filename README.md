# Chess Simulator
A terminal-based chess simulator written in Python.

## Features
- Memorize movement
- Standard chess piece movement
- Illegal move detection
- Check detection
- Checkmate and stalemate detection
- Castling
- Pawn promotion
- En passant
- Terminal-based gameplay

## How to Run

python main.py

## How to Play

enter movement commands in this format:
12,34
This means moving the piece at row 1 column 2 to row 3 column 4.
Type help during the game to see the input format again.
Type reverse to reverse game to a specific round.

## Current limitations
- The game runs only in the terminal
- The interface is text-based
- Some edge cases may still need further testing

## Project status
The main rule system is implemented and the game can run through a full chess match in the terminal.
